Let's start with the famous hello world! brick allows you to substitute variables into a template string or a dom element as following:

<div class="brick">
    <style>
      .brick {
          color: {{ color }};
      }
    </style>
    <span class="brick">{{ label }}</span>
</div>

Nothing more simple! 

var view = brick(tmpl).build();
view.set('label', 'Hello');

Tips: type hello color blue or hello label world in the console to update the demo!
